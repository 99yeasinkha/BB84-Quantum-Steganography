# BB84-Based Quantum-Secured Multimedia-in-Video Steganography System

## Overview

This research project proposes a BB84-based quantum-secured multimedia steganography framework for secure multimedia transmission.

The system integrates Quantum Key Distribution (QKD) with multimedia steganography to improve confidentiality and resistance against eavesdropping attacks.

## Research Objectives

- Secure multimedia transmission
- Quantum key generation using BB84 protocol
- Steganographic embedding
- Enhanced resistance against interception attacks

## Methodology

1. BB84 Quantum Key Distribution
2. Secret key generation
3. Multimedia steganographic embedding
4. Secure extraction and verification

## Results

- PSNR: 55.91 dB
- SSIM: 1.0000
- BER: 0.00000
